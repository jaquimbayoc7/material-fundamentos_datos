Correlación Estadística

Aprende a medir la fuerza y dirección de la relación entre variables numéricas. La correlación es esencial para comprender patrones en datos de calidad de agua, medio ambiente e ingeniería.

1. ¿Qué es la Correlación?

La correlación mide el grado de asociación lineal (o monotónica) entre dos variables cuantitativas. Su valor oscila entre −1 y +1.

Correlación: Medida estadística que cuantifica la intensidad y dirección de la relación entre dos variables. Un valor cercano a +1 indica relación positiva fuerte, cercano a −1 indica relación negativa fuerte, y cercano a 0 indica ausencia de relación lineal.

Interpretación del Coeficiente

Rango de |r| Interpretación Ejemplo
0.00 – 0.19 Correlación muy débil pH vs. Turbidez en agua potable
0.20 – 0.39 Correlación débil Sólidos disueltos vs. Cloraminas
0.40 – 0.59 Correlación moderada Conductividad vs. Sólidos totales
0.60 – 0.79 Correlación fuerte Temperatura vs. Oxígeno disuelto
0.80 – 1.00 Correlación muy fuerte Dureza vs. Calcio disuelto

💡 Correlación ≠ Causalidad

Que dos variables estén correlacionadas no significa que una cause la otra. Puede haber una tercera variable (confusora) que explique la relación. Ejemplo: la correlación entre el pH y la conductividad del agua podría deberse a la concentración de minerales disueltos.

2. Correlación de Pearson

Mide la relación lineal entre dos variables cuantitativas continuas. Es la más utilizada y asume distribución normal de los datos.

r = Σ(xᵢ - x̄)(yᵢ - ȳ) / √[Σ(xᵢ - x̄)² · Σ(yᵢ - ȳ)²]

Supuestos de Pearson

Linealidad

La relación entre las variables debe ser lineal. Verificar con un diagrama de dispersión.

Normalidad

Ambas variables deben distribuirse normalmente. Verificar con Shapiro-Wilk o histogramas.

Homoscedasticidad

La variabilidad de Y debe ser constante en todos los niveles de X.

Datos continuos

Ambas variables deben estar medidas en escala de intervalo o razón.

📊 Ejemplo: Correlación entre pH y Dureza del agua

import pandas as pd
from scipy import stats

# Cargar datos
url = "https://raw.githubusercontent.com/jaquimbayoc7/material-fundamentos_datos/main/data/water_potability.csv"
df = pd.read_csv(url)

# Eliminar valores nulos
df_clean = df[['ph', 'Hardness']].dropna()

# Calcular correlación de Pearson
r, p_value = stats.pearsonr(df_clean['ph'], df_clean['Hardness'])
print(f"Correlación de Pearson (r): {r:.4f}")
print(f"Valor p: {p_value:.4f}")
print(f"Interpretación: {'Significativa' if p_value < 0.05 else 'No significativa'}")

Interpretación: Si r ≈ 0.05 con p > 0.05, concluimos que no hay relación lineal significativa entre el pH y la dureza del agua.

📊 Ejemplo: Correlación Conductividad vs. Sólidos

# Correlación entre Conductividad y Sólidos totales
df_clean2 = df[['Conductivity', 'Solids']].dropna()

r2, p2 = stats.pearsonr(df_clean2['Conductivity'], df_clean2['Solids'])
print(f"r = {r2:.4f}, p = {p2:.4f}")

# Coeficiente de determinación
r_squared = r2 ** 2
print(f"R² = {r_squared:.4f}")
print(f"La conductividad explica el {r_squared*100:.1f}% de la variabilidad en sólidos")

El R² (coeficiente de determinación) indica qué porcentaje de la variabilidad de una variable es explicado por la otra.

3. Correlación de Spearman

Mide la relación monotónica (no necesariamente lineal) entre dos variables. Trabaja con los rangos de los datos, no con los valores originales.

ρ = 1 - (6 · Σdᵢ²) / [n(n² - 1)]
donde dᵢ = diferencia entre los rangos de cada par de observaciones

¿Cuándo usar Spearman?

Datos no normales

Cuando las variables no cumplen el supuesto de normalidad requerido por Pearson.

Variables ordinales

Cuando al menos una variable es ordinal (ej: nivel de contaminación: bajo, medio, alto).

Relaciones no lineales

Cuando la relación es monotónica pero no lineal (ej: exponencial, logarítmica).

Datos con outliers

Es más robusta a valores extremos que Pearson, ya que trabaja con rangos.

📊 Ejemplo: Spearman entre Turbidez y Carbono Orgánico

from scipy import stats

df_clean3 = df[['Turbidity', 'Organic_carbon']].dropna()

# Correlación de Spearman
rho, p_value = stats.spearmanr(df_clean3['Turbidity'], df_clean3['Organic_carbon'])
print(f"Correlación de Spearman (ρ): {rho:.4f}")
print(f"Valor p: {p_value:.4f}")

# Comparar con Pearson
r_pearson, _ = stats.pearsonr(df_clean3['Turbidity'], df_clean3['Organic_carbon'])
print(f"\nPearson: {r_pearson:.4f} vs Spearman: {rho:.4f}")
print("Si difieren mucho, la relación puede ser no lineal")

Cálculo paso a paso

muestra = {'Turbidez': [4.2, 2.1, 5.8, 1.3, 3.7],

4. Correlación de Kendall (τ)

Mide la asociación ordinal entre dos variables contando pares concordantes y discordantes. Es más robusta que Spearman para muestras pequeñas.

τ = (C - D) / [n(n-1)/2]
C = pares concordantes, D = pares discordantes
Par concordante: Si xᵢ > xⱼ y yᵢ > yⱼ (o ambos menores). Ambas variables se mueven en la misma dirección.
Par discordante: Si xᵢ > xⱼ pero yᵢ < yⱼ (o viceversa). Las variables se mueven en dirección opuesta.

📊 Ejemplo: Kendall entre Sulfato y Cloraminas

from scipy import stats

df_clean4 = df[['Sulfate', 'Chloramines']].dropna()

# Correlación de Kendall
tau, p_value = stats.kendalltau(df_clean4['Sulfate'], df_clean4['Chloramines'])
print(f"Correlación de Kendall (τ): {tau:.4f}")
print(f"Valor p: {p_value:.4f}")

# 4 muestras de agua: Sulfato y Cloraminas
# Muestra  Sulfato  Cloraminas
#    A       320       7.2
#    B       280       6.8
#    C       350       7.5
#    D       300       7.0

# Ordenamos por Sulfato: B(280), D(300), A(320), C(350)
# Cloraminas correspondientes: 6.8, 7.0, 7.2, 7.5

# Comparamos todos los pares (n=4 → 6 pares):
# (B,D): 280<300, 6.8<7.0 → Concordante ✓
# (B,A): 280<320, 6.8<7.2 → Concordante ✓
# (B,C): 280<350, 6.8<7.5 → Concordante ✓
# (D,A): 300<320, 7.0<7.2 → Concordante ✓
# (D,C): 300<350, 7.0<7.5 → Concordante ✓
# (A,C): 320<350, 7.2<7.5 → Concordante ✓

# C = 6, D = 0
# τ = (6 - 0) / (4×3/2) = 6/6 = 1.0
# Correlación perfecta positiva

5. Comparación de los Tres Métodos

Característica Pearson (r) Spearman (ρ) Kendall (τ)
Tipo de relación Lineal Monotónica Monotónica
Tipo de datos Continuos, normales Continuos u ordinales Continuos u ordinales
Sensibilidad a outliers Alta Baja Baja
Supuesto de normalidad Sí No No
Muestras pequeñas Menos preciso Bueno Mejor
Velocidad de cálculo Rápido — O(n) Moderado — O(n log n) Lento — O(n²)
Cuándo usar Datos normales, relación lineal Datos no normales, outliers Muestras pequeñas, datos ordinales

📊 Ejemplo: Comparar los 3 métodos sobre los mismos datos

from scipy import stats
import pandas as pd

url = "https://raw.githubusercontent.com/jaquimbayoc7/material-fundamentos_datos/main/data/water_potability.csv"
df = pd.read_csv(url)

variables = ['ph', 'Hardness', 'Chloramines', 'Sulfate', 'Conductivity',
             'Organic_carbon', 'Trihalomethanes', 'Turbidity']

# Ejemplo con ph vs Conductivity
x = df['ph'].dropna()
y = df['Conductivity'].loc[x.index].dropna()
common = x.index.intersection(y.index)
x, y = x.loc[common], y.loc[common]

r_pearson, p1  = stats.pearsonr(x, y)
r_spearman, p2 = stats.spearmanr(x, y)
r_kendall, p3  = stats.kendalltau(x, y)

print(f"Pearson:  r = {r_pearson:.4f}  (p = {p1:.4f})")
print(f"Spearman: ρ = {r_spearman:.4f}  (p = {p2:.4f})")
print(f"Kendall:  τ = {r_kendall:.4f}  (p = {p3:.4f})")
print()
print("Si Pearson ≈ Spearman → relación es lineal")
print("Si Spearman > Pearson → relación es no lineal pero monotónica")
print("Si todos ≈ 0 → no hay relación")

💡 ¿Cuál elegir?

Como regla general: comienza con Pearson. Si los datos no son normales o hay outliers, usa Spearman. Si la muestra es pequeña (n < 30), prefiere Kendall.

6. Matrices de Correlación

Una matriz de correlación muestra los coeficientes de correlación entre todos los pares de variables de un dataset. Es una herramienta fundamental para el análisis exploratorio de datos.

Matriz de Correlación: Tabla cuadrada simétrica donde cada celda (i, j) contiene el coeficiente de correlación entre la variable i y la variable j. La diagonal siempre vale 1 (correlación de una variable consigo misma).

Propiedades de la Matriz

Simétrica

corr(X, Y) = corr(Y, X). La mitad superior es espejo de la inferior.

Diagonal = 1

Toda variable tiene correlación perfecta consigo misma.

Valores entre -1 y 1

Todos los valores están acotados en el rango [-1, 1].

Semidefinida positiva

Todos sus autovalores son ≥ 0 (propiedad matemática importante).

📊 Ejemplo: Construir la matriz de correlación del dataset

import pandas as pd

url = "https://raw.githubusercontent.com/jaquimbayoc7/material-fundamentos_datos/main/data/water_potability.csv"
df = pd.read_csv(url)

# Matriz de Pearson
corr_pearson = df.corr(method='pearson')
print("=== Matriz de Correlación de Pearson ===")
print(corr_pearson.round(3))

# Matriz de Spearman
corr_spearman = df.corr(method='spearman')
print("\n=== Matriz de Correlación de Spearman ===")
print(corr_spearman.round(3))

# Encontrar las correlaciones más fuertes (excluyendo la diagonal)
import numpy as np
mask = np.triu(np.ones_like(corr_pearson, dtype=bool), k=1)
pares = corr_pearson.where(mask).stack().sort_values(ascending=False)
print("\n=== Top 5 correlaciones más fuertes ===")
print(pares.head())

📊 Ejemplo: Filtrar correlaciones significativas

# Encontrar pares con correlación > 0.3 (moderada o superior)
umbral = 0.3
fuertes = pares[pares.abs() > umbral]
print(f"Pares con |r| > {umbral}:")
for (var1, var2), valor in fuertes.items():
    print(f"  {var1} ↔ {var2}: r = {valor:.4f}")

# Si no hay pares fuertes, las variables son bastante independientes
if len(fuertes) == 0:
    print("No se encontraron correlaciones fuertes.")
    print("Esto significa que las variables son relativamente independientes.")

7. Visualización de Correlaciones

Los gráficos son esenciales para interpretar correlaciones. Aquí se presentan las técnicas más utilizadas.

Mapa de Calor (Heatmap)

📊 Ejemplo: Heatmap de la matriz de correlación

import seaborn as sns
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(10, 8))

corr_matrix = df.corr(method='pearson')

sns.heatmap(corr_matrix,
            annot=True,          # Mostrar valores
            fmt='.2f',           # 2 decimales
            cmap='RdBu_r',      # Rojo=negativo, Azul=positivo
            center=0,            # Centro en 0
            vmin=-1, vmax=1,     # Rango completo
            square=True,         # Celdas cuadradas
            linewidths=0.5,      # Líneas entre celdas
            ax=ax)

ax.set_title('Matriz de Correlación — Calidad del Agua', fontsize=14, pad=20)
plt.tight_layout()
plt.show()

Lectura del heatmap: Colores azules intensos = correlación positiva fuerte. Colores rojos intensos = correlación negativa fuerte. Colores claros = correlación débil.

Diagrama de Dispersión (Scatter Plot)

📊 Ejemplo: Scatter plot con línea de tendencia

import seaborn as sns
import matplotlib.pyplot as plt

fig, axes = plt.subplots(1, 3, figsize=(18, 5))

# Par 3: Chloramines vs Turbidity
sns.regplot(data=df, x='Chloramines', y='Turbidity', ax=axes[2],
            scatter_kws={'alpha': 0.3, 's': 10},
            line_kws={'color': 'red'})
axes[2].set_title('Cloraminas vs Turbidez')

plt.tight_layout()
plt.show()

Pair Plot (Gráfico de Pares)

📊 Ejemplo: Pairplot coloreado por Potabilidad

import seaborn as sns

# Seleccionar variables de interés
vars_interes = ['ph', 'Hardness', 'Chloramines', 'Turbidity', 'Potability']

sns.pairplot(df[vars_interes].dropna(),
             hue='Potability',       # Color por clase
             palette='Set1',
             diag_kind='kde',        # Densidad en la diagonal
             plot_kws={'alpha': 0.4, 's': 15})

plt.suptitle('Pair Plot — Variables de Calidad del Agua', y=1.02)
plt.show()

El pairplot permite ver simultáneamente la distribución individual de cada variable (diagonal) y la relación entre cada par (fuera de la diagonal).

9. Resumen y Puntos Clave

Pearson

Para relaciones lineales con datos normales. El más utilizado pero sensible a outliers.

Spearman

Para relaciones monotónicas. Robusto a outliers. Ideal cuando no hay normalidad.

Kendall

Basado en pares concordantes. Mejor para muestras pequeñas y datos ordinales.

Matrices + Heatmaps

Visión global de todas las relaciones. Esencial en análisis exploratorio.

🎯 Recuerda

La correlación cuantifica la asociación, no la causalidad. Siempre complementa los análisis numéricos con visualizaciones. Un coeficiente estadísticamente significativo puede no ser prácticamente relevante si el tamaño del efecto es pequeño.
