Supuestos Estadísticos y Pruebas de Hipótesis

Domina los supuestos que deben cumplirse antes de aplicar modelos estadísticos y aprende a validar o rechazar hipótesis con datos reales de calidad del agua.

1. ¿Por Qué Importan los Supuestos?

Los modelos estadísticos (regresión, ANOVA, etc.) se basan en supuestos matemáticos. Si estos no se cumplen, los resultados pueden ser incorrectos o engañosos.

Supuestos Estadísticos: Condiciones que deben cumplirse para que un modelo o prueba estadística produzca resultados válidos y confiables. Violarlos puede llevar a conclusiones erróneas.

Supuestos principales en modelos de regresión

Linealidad

La relación entre variables independientes y dependiente es lineal.

Independencia

Las observaciones son independientes entre sí (no autocorrelación).

Homocedasticidad

La varianza de los errores es constante para todos los niveles de X.

Normalidad de residuos

Los errores del modelo siguen una distribución normal.

No multicolinealidad

Las variables independientes no están altamente correlacionadas entre sí.

2. Homocedasticidad

Es el supuesto de que la varianza de los errores (residuos) es constante a lo largo de todos los valores de la variable independiente.

Homocedasticidad: Del griego "homo" (igual) + "skedasis" (dispersión). Significa que la dispersión de los residuos es uniforme. Si graficas los residuos vs. los valores predichos, deberían distribuirse en una banda horizontal uniforme.

¿Cómo se ve gráficamente?

✅ Homocedasticidad (Correcto)

Los residuos forman una nube uniforme sin patrón alrededor del eje cero.

La dispersión es similar para valores bajos y altos de X.

📊 Ejemplo: Verificar homocedasticidad visualmente

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

url = "https://raw.githubusercontent.com/jaquimbayoc7/material-fundamentos_datos/main/data/water_potability.csv"
df = pd.read_csv(url).dropna()

# Modelo simple: predecir Solids con Conductivity
X = df[['Conductivity']].values
y = df['Solids'].values

modelo = LinearRegression()
modelo.fit(X, y)
y_pred = modelo.predict(X)
residuos = y - y_pred

# Gráfico de residuos vs valores predichos
plt.figure(figsize=(10, 5))
plt.scatter(y_pred, residuos, alpha=0.3, s=10)
plt.axhline(y=0, color='red', linestyle='--', linewidth=2)
plt.xlabel('Valores Predichos')
plt.ylabel('Residuos')
plt.title('Gráfico de Residuos vs Predichos')
plt.show()

# Si la nube es uniforme → homocedasticidad ✓
# Si hay forma de embudo → heterocedasticidad ✗

Prueba formal: Breusch-Pagan

La prueba de Breusch-Pagan evalúa estadísticamente si la varianza de los residuos depende de las variables independientes.

H₀: La varianza de los errores es constante (homocedasticidad)
H₁: La varianza de los errores NO es constante (heterocedasticidad)

Si p-valor < 0.05 → Rechazar H₀ → Hay heterocedasticidad

📊 Ejemplo: Prueba de Breusch-Pagan

import statsmodels.api as sm
from statsmodels.stats.diagnostic import het_breuschpagan

# Ajustar modelo OLS
X_const = sm.add_constant(df[['Conductivity']])
modelo_ols = sm.OLS(df['Solids'], X_const).fit()
residuos = modelo_ols.resid

# Prueba de Breusch-Pagan
lm_stat, lm_pvalue, f_stat, f_pvalue = het_breuschpagan(residuos, X_const)

print(f"Estadístico LM: {lm_stat:.4f}")
print(f"Valor p (LM):   {lm_pvalue:.4f}")
print(f"Estadístico F:  {f_stat:.4f}")
print(f"Valor p (F):    {f_pvalue:.4f}")
print()
if lm_pvalue < 0.05:
    print("⚠️ Se rechaza H₀: HAY heterocedasticidad")
    print("   Los residuos no tienen varianza constante")
else:
    print("✅ No se rechaza H₀: hay homocedasticidad")
    print("   Los residuos tienen varianza constante")

Prueba de Goldfeld-Quandt

Compara la varianza de los residuos en dos subgrupos del dataset (valores bajos vs. altos de X).

📊 Ejemplo: Prueba de Goldfeld-Quandt

from statsmodels.stats.diagnostic import het_goldfeldquandt

# Prueba de Goldfeld-Quandt
f_stat, p_value, ordering = het_goldfeldquandt(
    df['Solids'].dropna(),
    sm.add_constant(df[['Conductivity']]).dropna()
)

print(f"Estadístico F: {f_stat:.4f}")
print(f"Valor p:       {p_value:.4f}")
print()
if p_value < 0.05:
    print("⚠️ Hay heterocedasticidad (varianza distinta en subgrupos)")
else:
    print("✅ Homocedasticidad (varianza similar en subgrupos)")

💡 ¿Qué hacer si hay heterocedasticidad?

  • Transformar la variable dependiente: log(y), √y, 1/y
  • Usar errores estándar robustos: modelo.fit(cov_type='HC3') en statsmodels
  • Mínimos Cuadrados Ponderados (WLS): Dar menos peso a observaciones con mayor varianza
  • Regresión robusta: Usar modelos que no asuman homocedasticidad

3. Heterocedasticidad: Causas y Consecuencias

Cuando la varianza de los errores no es constante, estamos ante heterocedasticidad. Es importante entender por qué ocurre y qué efectos tiene.

Causas comunes

Efecto escala

Variables con valores grandes tienden a tener mayor variabilidad. Ej: la variabilidad de sólidos totales es mayor en aguas con alta concentración.

Datos agrupados

Diferentes subgrupos tienen diferentes niveles de variabilidad. Ej: agua potable vs. no potable pueden tener varianza distinta.

Modelo mal especificado

Falta incluir una variable relevante o la forma funcional es incorrecta (debería ser cuadrática, no lineal).

Outliers

Valores extremos en ciertas zonas del rango pueden inflar la varianza local de los residuos.

Consecuencias

⚠️ Efectos de la heterocedasticidad en regresión

  • Coeficientes: Siguen siendo insesgados (no cambian), pero ya no son eficientes (no tienen la menor varianza posible).
  • Errores estándar: Son incorrectos, generalmente subestimados.
  • Pruebas de hipótesis: Los valores t y F se distorsionan → puedes rechazar H₀ cuando no deberías (más falsos positivos).
  • Intervalos de confianza: Son más estrechos de lo que deberían ser.

📊 Ejemplo: Solución con transformación logarítmica

import numpy as np
import statsmodels.api as sm
from statsmodels.stats.diagnostic import het_breuschpagan

# Modelo original
X = sm.add_constant(df[['Conductivity']].dropna())
y = df['Solids'].loc[X.index]

modelo1 = sm.OLS(y, X).fit()
_, p1, _, _ = het_breuschpagan(modelo1.resid, X)
print(f"Modelo original  → BP p-valor: {p1:.4f}")

# Modelo con log(y)
y_log = np.log(y)
modelo2 = sm.OLS(y_log, X).fit()
_, p2, _, _ = het_breuschpagan(modelo2.resid, X)
print(f"Modelo log(y)    → BP p-valor: {p2:.4f}")

# Si p2 > p1, la transformación mejoró la homocedasticidad

4. Multicolinealidad y Factor de Inflación de Varianza (VIF)

La multicolinealidad ocurre cuando dos o más variables independientes están altamente correlacionadas entre sí, lo que dificulta distinguir el efecto individual de cada una.

Multicolinealidad: Situación en la que las variables predictoras de un modelo de regresión múltiple están correlacionadas entre sí. Esto infla la varianza de los coeficientes estimados y hace que sean inestables.

¿Cómo detectarla?

Matriz de Correlación

Correlaciones > 0.7 (en valor absoluto) entre predictores sugieren multicolinealidad.

Limitación: Solo detecta relaciones entre pares de variables.

VIF (Factor de Inflación de Varianza)

Mide cuánto se infla la varianza de un coeficiente por la correlación con los demás predictores.

Ventaja: Detecta multicolinealidad entre múltiples variables simultáneamente.

Factor de Inflación de Varianza (VIF)

VIF(xⱼ) = 1 / (1 - Rⱼ²)
donde Rⱼ² es el R² de regresionar xⱼ contra todas las demás variables independientes

Interpretación del VIF

Valor VIF Interpretación Acción
1 Sin correlación con otras variables Ideal, ninguna acción necesaria
1 – 5 Correlación moderada Generalmente aceptable
5 – 10 Correlación alta Investigar y posiblemente eliminar variables
> 10 Multicolinealidad severa Definitivamente eliminar o combinar variables

📊 Ejemplo: Calcular VIF para todas las variables

import pandas as pd
from statsmodels.stats.outliers_influence import variance_inflation_factor
import statsmodels.api as sm

url = "https://raw.githubusercontent.com/jaquimbayoc7/material-fundamentos_datos/main/data/water_potability.csv"
df = pd.read_csv(url).dropna()

# Variables independientes (excluir Potability que sería la dependiente)
variables = ['ph', 'Hardness', 'Solids', 'Chloramines', 'Sulfate',
             'Conductivity', 'Organic_carbon', 'Trihalomethanes', 'Turbidity']

X = df[variables]
X_const = sm.add_constant(X)

# Calcular VIF para cada variable
vif_data = pd.DataFrame()
vif_data['Variable'] = variables
vif_data['VIF'] = [variance_inflation_factor(X_const.values, i+1)
                   for i in range(len(variables))]

# Ordenar por VIF descendente
vif_data = vif_data.sort_values('VIF', ascending=False)
print("=== Factor de Inflación de Varianza (VIF) ===")
print(vif_data.to_string(index=False))
print()
for _, row in vif_data.iterrows():
    if row['VIF'] > 10:
        print(f"⚠️ {row['Variable']}: VIF = {row['VIF']:.2f} → Multicolinealidad SEVERA")
    elif row['VIF'] > 5:
        print(f"⚠️ {row['Variable']}: VIF = {row['VIF']:.2f} → Multicolinealidad alta")
    else:
        print(f"✅ {row['Variable']}: VIF = {row['VIF']:.2f} → Aceptable")

¿Qué hacer si hay multicolinealidad?

Eliminar variables

Remover la variable con mayor VIF y recalcular. Repetir hasta que todos los VIF sean < 5.

Combinar variables

Usar PCA (Análisis de Componentes Principales) para crear nuevas variables no correlacionadas.

Regularización

Usar Ridge o Lasso regression, que penalizan coeficientes grandes y reducen el efecto de la multicolinealidad.

Más datos

Aumentar el tamaño de la muestra puede reducir la varianza de los coeficientes, aunque no elimina la correlación.

📊 Ejemplo: Eliminar variable con mayor VIF iterativamente

from statsmodels.stats.outliers_influence import variance_inflation_factor
import statsmodels.api as sm

def calcular_vif(dataframe, variables):
    """Calcula VIF para un conjunto de variables."""
    X = sm.add_constant(dataframe[variables])
    vif = pd.DataFrame({
        'Variable': variables,
        'VIF': [variance_inflation_factor(X.values, i+1) for i in range(len(variables))]
    })
    return vif.sort_values('VIF', ascending=False)

# Proceso iterativo
vars_actuales = variables.copy()
umbral_vif = 5.0

print("=== Eliminación iterativa por VIF ===")
while True:
    vif_df = calcular_vif(df, vars_actuales)
    max_vif = vif_df['VIF'].max()

    if max_vif <= umbral_vif:
        print(f"\n✅ Todos los VIF ≤ {umbral_vif}")
        print(f"Variables finales: {vars_actuales}")
        break

    var_eliminar = vif_df.iloc[0]['Variable']
    print(f"Eliminando {var_eliminar} (VIF = {max_vif:.2f})")
    vars_actuales.remove(var_eliminar)

print("\nVIF final:")
print(calcular_vif(df, vars_actuales).to_string(index=False))

5. Pruebas de Hipótesis: Fundamentos

Las pruebas de hipótesis son el mecanismo formal para tomar decisiones basadas en datos. Permiten determinar si un patrón observado es real o solo efecto del azar.

Prueba de Hipótesis: Procedimiento estadístico que utiliza datos muestrales para evaluar una afirmación sobre una población. Se basa en la probabilidad de observar los datos bajo el supuesto de que la afirmación (hipótesis nula) es verdadera.

Conceptos clave

Hipótesis Nula (H₀)

Afirmación de "no efecto" o "no diferencia". Es lo que asumimos como verdadero hasta que la evidencia diga lo contrario.

Ej: "La media de pH no difiere entre agua potable y no potable."

Hipótesis Alternativa (H₁)

Afirmación contraria a H₀. Es lo que queremos demostrar con evidencia estadística.

Ej: "La media de pH SÍ difiere entre agua potable y no potable."

Nivel de Significancia (α)

Probabilidad máxima de rechazar H₀ cuando es verdadera (error tipo I). Comúnmente α = 0.05 (5%).

Valor p (p-value)

Probabilidad de obtener un resultado tan extremo o más, asumiendo que H₀ es verdadera. Si p < α, rechazamos H₀.

Tipos de errores

H₀ es Verdadera H₀ es Falsa
No rechazar H₀ ✅ Decisión correcta ❌ Error Tipo II (β) — Falso negativo
Rechazar H₀ ❌ Error Tipo I (α) — Falso positivo ✅ Decisión correcta (Potencia = 1 − β)

💡 Analogía del juicio

Piensa en un juicio: H₀ = "el acusado es inocente". Error Tipo I = condenar a un inocente. Error Tipo II = absolver a un culpable. El sistema judicial prefiere Error Tipo II (es peor condenar a un inocente), por eso α se mantiene bajo (0.05).

Pasos de una prueba de hipótesis

Paso 1: Plantear hipótesis

Definir H₀ y H₁ claramente. H₀ siempre contiene un signo de igualdad (=, ≤, ≥).

Paso 2: Elegir α

Definir el nivel de significancia. Lo más común es α = 0.05 (5% de probabilidad de error tipo I).

Paso 3: Calcular estadístico

Aplicar la fórmula de la prueba (t, F, χ²) para obtener el estadístico de prueba.

Paso 4: Decidir

Comparar p-valor con α. Si p < α → rechazar H₀. Si p ≥ α → no rechazar H₀.

6. Pruebas de Hipótesis Comunes

6.1 Prueba t de Student (dos muestras independientes)

Compara las medias de dos grupos independientes para determinar si difieren significativamente.

H₀: μ₁ = μ₂ (las medias son iguales)
H₁: μ₁ ≠ μ₂ (las medias son diferentes)

📊 Ejemplo: ¿Difiere el pH entre agua potable y no potable?

from scipy import stats
import pandas as pd

url = "https://raw.githubusercontent.com/jaquimbayoc7/material-fundamentos_datos/main/data/water_potability.csv"
df = pd.read_csv(url)

# Separar grupos
ph_potable = df[df['Potability'] == 1]['ph'].dropna()
ph_no_potable = df[df['Potability'] == 0]['ph'].dropna()

# Prueba t de dos muestras independientes
t_stat, p_value = stats.ttest_ind(ph_potable, ph_no_potable)

print("=== Prueba t de Student ===")
print(f"H₀: μ_potable = μ_no_potable")
print(f"H₁: μ_potable ≠ μ_no_potable")
print(f"α = 0.05")
print(f"")
print(f"Media pH potable:    {ph_potable.mean():.4f}")
print(f"Media pH no potable: {ph_no_potable.mean():.4f}")
print(f"Estadístico t:       {t_stat:.4f}")
print(f"Valor p:             {p_value:.4f}")
print()
if p_value < 0.05:
    print("✅ Se rechaza H₀: las medias de pH son significativamente diferentes")
else:
    print("❌ No se rechaza H₀: no hay evidencia de diferencia significativa en pH")

6.2 Prueba de correlación de Pearson

Evalúa si la correlación lineal entre dos variables es estadísticamente significativa.

H₀: ρ = 0 (no hay correlación lineal)
H₁: ρ ≠ 0 (hay correlación lineal)

📊 Ejemplo: ¿Hay correlación significativa entre Cloraminas y Turbidez?

from scipy import stats

df_clean = df[['Chloramines', 'Turbidity']].dropna()

r, p_value = stats.pearsonr(df_clean['Chloramines'], df_clean['Turbidity'])

print("=== Prueba de correlación de Pearson ===")
print(f"H₀: ρ = 0 (no hay correlación)")
print(f"H₁: ρ ≠ 0 (hay correlación)")
print(f"")
print(f"Coeficiente r: {r:.4f}")
print(f"Valor p:       {p_value:.4f}")
print()
if p_value < 0.05:
    direccion = "positiva" if r > 0 else "negativa"
    print(f"✅ Correlación {direccion} significativa (r={r:.4f}, p={p_value:.4f})")
else:
    print("❌ No hay correlación significativa")

6.3 ANOVA de un factor

Compara las medias de tres o más grupos simultáneamente. Es una extensión de la prueba t.

H₀: μ₁ = μ₂ = μ₃ = ... = μₖ (todas las medias son iguales)
H₁: Al menos una media es diferente

📊 Ejemplo: ANOVA — ¿El pH varía según el nivel de dureza?

from scipy import stats
import pandas as pd

# Crear grupos de dureza: Baja, Media, Alta
df_anova = df[['ph', 'Hardness']].dropna()
df_anova['Grupo_Dureza'] = pd.cut(df_anova['Hardness'],
                                   bins=3,
                                   labels=['Baja', 'Media', 'Alta'])

# Separar pH por grupo
grupos = [grupo['ph'].values for _, grupo in df_anova.groupby('Grupo_Dureza')]

# ANOVA de un factor
f_stat, p_value = stats.f_oneway(*grupos)

print("=== ANOVA de un factor ===")
print(f"H₀: μ_baja = μ_media = μ_alta")
print(f"H₁: Al menos una media difiere")
print(f"")
print("Medias por grupo:")
for nombre, grupo in df_anova.groupby('Grupo_Dureza'):
    print(f"  Dureza {nombre}: pH medio = {grupo['ph'].mean():.4f}")
print(f"")
print(f"Estadístico F: {f_stat:.4f}")
print(f"Valor p:       {p_value:.4f}")
print()
if p_value < 0.05:
    print("✅ Se rechaza H₀: al menos un grupo tiene media diferente")
    print("   Siguiente paso: hacer prueba post-hoc (Tukey) para saber cuál")
else:
    print("❌ No se rechaza H₀: no hay diferencia significativa entre grupos")

6.4 Prueba de Shapiro-Wilk (normalidad)

Evalúa si una muestra proviene de una distribución normal. Es un supuesto previo para muchas pruebas.

📊 Ejemplo: ¿El pH sigue una distribución normal?

from scipy import stats
import matplotlib.pyplot as plt

ph_data = df['ph'].dropna()

# Prueba de Shapiro-Wilk (usar muestra si n > 5000)
muestra = ph_data.sample(min(5000, len(ph_data)), random_state=42)
w_stat, p_value = stats.shapiro(muestra)

print("=== Prueba de Shapiro-Wilk ===")
print(f"H₀: Los datos siguen una distribución normal")
print(f"H₁: Los datos NO siguen una distribución normal")
print(f"")
print(f"Estadístico W: {w_stat:.4f}")
print(f"Valor p:       {p_value:.4f}")
print()
if p_value < 0.05:
    print("⚠️ Se rechaza H₀: los datos NO son normales")
    print("   Usar pruebas no paramétricas (Spearman, Mann-Whitney)")
else:
    print("✅ No se rechaza H₀: los datos son compatibles con normalidad")
    print("   Se pueden usar pruebas paramétricas (Pearson, t-test)")

# Visualización
fig, axes = plt.subplots(1, 2, figsize=(12, 5))
axes[0].hist(ph_data, bins=40, edgecolor='black', alpha=0.7)
axes[0].set_title('Histograma de pH')
stats.probplot(muestra, plot=axes[1])
axes[1].set_title('Q-Q Plot de pH')
plt.tight_layout()
plt.show()

8. Resumen y Puntos Clave

Homocedasticidad

Varianza constante de residuos. Verificar con gráficos y Breusch-Pagan. Si falla: transformar datos o usar errores robustos.

Multicolinealidad

Variables predictoras correlacionadas. Detectar con VIF. Si VIF > 5: eliminar o combinar variables.

Pruebas de Hipótesis

Mecanismo formal de decisión: H₀ vs H₁, calcular p-valor, comparar con α = 0.05.

Siempre verificar supuestos

Antes de aplicar cualquier prueba, verificar que se cumplen sus supuestos. Si no: usar alternativas no paramétricas.
